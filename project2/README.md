# Machine learning for quantum many-body problems

This repo contains a program that applies restricted boltzmann machine to the
quantum many body problem

## Usage

```console
$ cargo run [--release] -- [arguments]
```

```
OPTIONS:
    -d, --dimensions <DIMENSIONS>
            Number of dimensions in the simulation [default: 2]

        --delta <DELTA>
            Tolerance for the optimizer [default: 0.000000001]

        --delta-t <DELTA_T>
            Delta t [default: 0.5]

    -h, --help
            Print help information

        --hidden-layers <HIDDEN_LAYERS>
            Number of hidden layers [default: 2]

    -i, --interaction
            Whether to include interactions or not

        --learning-rate <LEARNING_RATE>
            Learning rate [default: 0.2]

        --max-optimizer-steps <MAX_OPTIMIZER_STEPS>
            Maximum number of optimizer steps [default: 100]

        --monte-carlo-cycles <MONTE_CARLO_CYCLES>
            Number of monte carlo cycles to run [default: 10000]

    -n, --num-particles <NUM_PARTICLES>
            Number of particles in the simulation [default: 2]

    -o, --optimizer <OPTIMIZER>
            The optimizer method [default: gradient-descent] [possible values: gradient-descent,
            adam]

    -o, --one-body-density
            Whether to include one body density or not

        --omega <OMEGA>
            Omega [default: 1]

    -r, --run-type <RUN_TYPE>
            What to run [default: normal] [possible values: normal, analysis]

    -s, --sampler <SAMPLER>
            The sampler method [default: brute-force] [possible values: brute-force,
            importance-sampling]

        --seed <SEED>
            Random seed to use [default: 42]

    -V, --version
            Print version information

        --verbose
            Wether to include many prints or not

        --weight-standard-deviation <WEIGHT_STANDARD_DEVIATION>
            Standard deviation for the weight initialization [default: 0.25]
```

## File structure

```
.
├── Cargo.toml                - contains required packages
├── report.pdf                - a paper containing theory and results
├── output                    - the outputs generated by the program
└── src                       - contains the rust and python source code
   ├── analysis.py            - for doing analysis and creating plots
   ├── args.rs                - for parsing arguments
   ├── blocking.py            - for doing statistical analysis with blocking
   ├── boltzmann.rs           - contains the main boltzmann system used in the simulations
   ├── main.rs                - contains the main script for running the different simulations
   ├── monte_carlo.rs         - contains the monte carlo methods
   ├── optimizer              - the different optimizers
   │  ├── adam.rs             - implementation of the ADAM optimization
   │  └── gradient_descent.rs - implementation of the gradient descent method
   ├── sampler                - the different sampling methods
   │  ├── brute_force.rs      - implementation of brute force sampling
   │  └── importance.rs       - implementation of importance sampling
   └── wave_function.rs       - contains the wave function methods of the system
```
